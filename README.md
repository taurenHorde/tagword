## '서비스 내용' 

소설을 다 같이 작성하면 어떨까? 생각에서 나온 다 같이 쓰는 소설
처음에는 끝말잇기 형태로 생각을 하였으나, 종결어미로만 끝나는 바람에
자유모드를 만듬. 문장의 추가는 socket으로 실시간 데이터를 주고받아
소설이 중간에 산으로 가지 않도록 방지!

좋은 문장엔 감정표현으로 표시 가능!
